# EQDmgAnalyzr

航空写真を利用した地震時における建物災害被害解析プログラム

## Environment
- Python 3.x ( >= 3.6)
- OpenCV 3.x
- NumPy
- SciPy
- Matplotlib
- scikit-learn
- tqdm

## Setup

```bash
./setup.sh
source ./venv/bin/activate
```
